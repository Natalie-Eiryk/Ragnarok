Origin

A story built to be read in the open.

Ragnarok is a reading-first web presence: chapters in one direction, lore in another, and a conversational line back into the world through Astrid. The site is meant to feel like a transmission archive, not a storefront or a locked box.

How it works

Three public paths

Read the chapters Published PDFs stay available, while future chapters render as honest coming-soon placeholders instead of broken embeds. Browse answered transmissions Astrid's archive now lives as static site data, so the public page no longer depends on live browser calls to GitHub. Explore the lore files Character, place, and timeline dossiers give the story more room to breathe between chapter drops.
Design intent

Free to read, built to return to.

Public surface

Readers should only hit story pages, lore pages, and the Astrid submission line. Operational tooling is no longer part of the public site map.

Private surface

LUMI and command tools stay available privately behind Cloudflare Access on the separate `lumi` host, instead of being mixed into the reading experience.
